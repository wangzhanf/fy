# CSS引入的三种方式

1 内嵌式：将css代码内嵌到html标签中（通过style属性定义）【不推荐】

```html
<h1 style="color: red;">HTML 表格</h1>
缺陷：  只能影响当前标签中的内容以及子标签中的内容
```

2 内联式：将css代码放置在head标签中的style标签

```html
<style type="text/css">
    * {
        font-size: 18px;
    }
</style>
缺陷：定义的样式只对当前文档有效
```

3 外联式： 创建独立的 ".css"文件，在页面中通过  link标签引入

# CSS选择器

目的：筛选元素应用样式

原因：如果仅使用基本选择器，为了实现复杂的控制需要在HTML代码添加无意义的标签，会破坏HTML的语义性

HTML5=HTML+CSS+JavaScript（库和框架）--》很多的库和框架都是使用css选择器的方式处理元素

## 选择器的分类

### 基本选择器

“ * ”   	匹配所有元素	一般用作页面和站点的初始化

“ E ”		匹配对应的HTML标签（标签选择器）      一般作为某个标签的初始化

“ #id ”	匹配的是标签中id属性      原则上一个页面上id是唯一的

" .class " 匹配的是标签中class属性    class可以重复

“选择器1，选择器N”      只要匹配任意一个选择器则应用该样式

### 层次选择器

"祖宗选择器   后代选择器" 	使用**空格**分隔     不考虑层次的嵌入关系

"父选择器>子选择器"    使用**>**分隔   必须是父子关系

"兄选择器+弟选择器"    使用+分隔    相邻兄弟选择器

"兄选择器~弟选择器"    使用~分隔	所有弟弟选择器

### 动态伪类选择器

伪类:根据元素的**状态**匹配

### 目标伪类选择器

通过给超链接添加链接到锚点,可以指定跳转到页面的指定位置

### UI元素状态伪类选择器

UI用户接口，web应用中用户和服务器交互的方式最主要的就是表单，UI就是表单元素，有些表单如单选、复选框都存在选中状态

### 结构伪类选择器

主要用于筛选结构中的子元素的位置,可以通过表达式表示奇数或偶数元素,用途一般作为表格,列表的展示

### 否定伪类选择器

取反

E：not(F)

选择器选择的基础是  E 筛选出不属于F的元素

### 伪元素选择器

根据元素所处的位置进行选择，

常见   ::first-letter    ::first-line     ::before    ::after

css新标准中可以将 ：：  简略写为   ：

其中 ：：before和：：after最常用

### 属性选择器

前面使用   .class     #id     这两个选择器就是属性

为了解决频繁使用class和id造成的代码冗余，利用了部分元素标签有一些特定的属性，通过属性区分和筛选元素【表单，href】

语法：【属性名 比较符  属性值】

[class="dis"],[class="disabled"]   ==>   [class~="dis"]

# 优先级问题

前提：CSS叫层叠样式表，显示的最终样式受所有的父容器控制，子容器会继承父容器的样式，如果样式没有冲突，如果样式出现冲突，按照就近原则（应用离自己最近容器定义的样式），如果出现了重名的样式，后面的生效。



## 优先级算法：

基本原则叫做     ABCD   算法

如果元素存在行内样式（内嵌样式）   				则A取值为    1

如果元素有相应的id选择器匹配           				则B取值为     1

如果元素有相应的class，以及伪类	  				则C加1

如果元素有相应的标签选择器，以及伪元素		则D加1





样式应用时，ABCD按照数值对比，数值大的最终生效     

1000   >   0123   >  0011



# 盒子模型

css的目的是布局和修饰HTML元素，HTML元素是一个容器，容器会占用空间【矩形空间】所以称之为盒子

页面中的容器都是盒子，所有的盒子共同布局了整个页面



## CSS中元素的类型：

块元素：默认情况独立成行，可以设定宽高，边距等		例如div  ul 			通过   display: block;

行内元素：和其他行内元素同行,默认没有宽高     例如span   a	 通过   display: inline;

行内块元素： 和其他的行内元素同行，可以设定宽高，边距等     例如  img  input   ，通过   display ：  inline-block;



如果控制元素的类型，使用display特性，扩展，如果display为none则隐藏元素

## 盒子的空间计算

盒子内部空间（内容空间）大小是通过width和height指定的

整个盒子的占用空间包括    width+padding+border+margin

## 盒子的浮动

文档流：  页面中的元素默认情况下按照源码的先后顺序{流}在视图窗口布局，源码在前，布局在前   。

浮动特性：   float        让容器脱离文档流

特性值：

​			left：左浮动，定位在父容器的左边或兄弟容器的右边

​			right：右浮动，定位在父容器的右边或兄弟容器的左边

清除浮动关系：

​	clear特性

# 定位

position   特性

使元素容器彻底脱离文档流，一般可以创造重叠的元素

元素可以使用的顶部，底部，左侧和右侧属性定位。然而，这些属性无法工作，除非事先设定 position 属性。一般使用两个坐标就可以定位

需要进行定位的元素一般都放在父容器的最后，防止元素定位影响后续元素的布局

常见的定位方式

static	是默认的   

fixed	附着，不随文档流移动

Relative 相对	

Absolute 定位
绝对定位的元素的位置相对于最近的已定位父元素，如果元素没有显式的已定位的父元素，那么它的位置相对于 < html>:	

如果使用绝对定位，父元素需要显式定位【父相对，子绝对】

# CSS隐藏元素

第1种方式:将高度设置为0,如果内部存在元素,元素本身不隐藏,一般需要结合溢出处理,让元素隐藏

第2种方式   ,让空间的透明度为0,空间依然占用

第3种方式, visibility特性, 空间依然占用

第4种方式,  display特性,空间被消失,有可能造成页面元素的抖动,一般都会将该区间设置为绝对定位
