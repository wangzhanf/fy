# 项目要求

开发一套能够兼容大部分设备的应用网关界面，监控各种设备的运行状态，可以通过接口调整各个设备的参数

# 考虑的因素

响应式页面【媒体响应+数据响应】

合理的使用框架解决：前端框架BootStrap + jQuery+VUE

SAP应用：单页面应用
